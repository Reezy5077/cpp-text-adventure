# Mini Text Adventure Game (C++)

This is a simple text-based adventure game written in C++.

## How it works:
The player makes choices that change the outcome of the story.

## Features:
- User input handling
- If/else logic
- Branching story paths

## What I learned:
- decision making in code
- nested conditions
- structuring a simple program

## Purpose:
This project was created to improve my programming skills and demonstrate logical thinking and interest in software development.
